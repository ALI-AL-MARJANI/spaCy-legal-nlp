
## Custom Extension: Legal NLP Assistant 

This fork includes a complete end-to-end NLP project for contract analysis:

- Clause classification  
- Legal NER (entities like PARTY, EFFECTIVE_DATE, PAYMENT_AMOUNT, etc..)  
- Custom risk scoring component integrated into the spaCy pipeline  
- FastAPI inference server  
- Full spaCy project structure (config, scripts, training workflow)
